### Docker
```
docker-compose up -d --build
```

### Instalação para executar sem Docker
Primeiro navegue para o diretório do front-end
```
cd pokedex
```
E depois instale as dependências

```
npm install
```

### Executar sem docker

```
npm run dev
```
Acesse em:
```
http://localhost:5173/
```

### Testes
```
npm run test
```
